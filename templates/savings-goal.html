üìö Bisection Method Explanation

The Bisection Method is a numerical root-finding algorithm that repeatedly divides an interval in half and selects the subinterval where the function changes sign, guaranteeing convergence.

How it Works

1. Start with an interval [a, b] where f(a) and f(b) have opposite signs
2. Calculate midpoint c = (a + b) / 2
3. Evaluate f(c)
4. Replace either a or b with c based on the sign of f(c)
5. Repeat until the interval is sufficiently small

c = (a + b) / 2

Application to Savings Goals

We convert the savings problem into finding the root of:
f(S) = S √ó [(1 + r)^n - 1] / r - TargetAmount
