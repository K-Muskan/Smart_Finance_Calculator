üìö Numerical Methods Explanation

This calculator uses numerical root-finding algorithms to solve the loan payment equation. Both methods iteratively improve approximations to find where a function equals zero.

Newton-Raphson Method

xn+1 = xn - f(xn) / f'(xn)

How it works: Uses the function value and its derivative at each point to generate better approximations. Requires computing the derivative explicitly.

Advantages: Quadratic convergence (very fast) when close to solution, typically converges in 5-8 iterations.

Requirements: Needs derivative f'(x) and one initial guess.

Secant Method

xn+1 = xn - f(xn) ¬∑ (xn - xn-1) / (f(xn) - f(xn-1))

How it works: Approximates the derivative using finite differences between two points. No explicit derivative calculation needed.

Advantages: Superlinear convergence, doesn't require derivative computation, generally reliable for most loan calculations.

Requirements: Needs two initial guesses.

Application to Loans

For loan calculations, we rearrange the payment equation P = M √ó [(1 - (1+r)^-n) / r] into the form f(M) = 0, where finding the root gives us the monthly payment M.
